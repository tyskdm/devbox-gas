# devbox-gas

development toolbox for google apps script.


## Version info:

version 0.1.0  create new.


## Usage

### install

in your project directory:

```shell
git submodule add git@github.com:tyskdm/devbox-gas.git ./devbox
```

### commands

#### up

```shell
./devbox/up
```

#### start

```shell
./devbox/start
```

#### stop

```shell
./devbox/stop
```

#### attach

```shell
./devbox/attach
```

#### exec

```shell
./devbox/exec COMMAND [ OPTIONS ]
```

#### delete

```shell
./devbox/delete
```
